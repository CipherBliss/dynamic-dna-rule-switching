# dynamic-dna-rule-switching
# 🧬 A Hybrid Entropy and Chaotic Map Based Dynamic DNA Rule Switching Approach for Secure Image Steganography

---

## 📌 Overview

This project presents a **secure image steganography technique** using a hybrid approach that combines:

* **Entropy-based randomness**
* **Chaotic maps for unpredictability**
* **Dynamic DNA rule switching**

The goal is to **securely hide data inside images** while ensuring high randomness and resistance against attacks.

---

## 🎯 Objective

* To develop a **secure image steganography system**
* To enhance security using **DNA encoding techniques**
* To dynamically switch DNA rules using **entropy and chaotic behavior**
* To prevent detection and improve robustness against attacks

---

## 🧠 Core Concepts Explained

### 🔹 DNA Encoding

Binary data is converted into DNA sequences using symbols:

* A (Adenine)
* T (Thymine)
* C (Cytosine)
* G (Guanine)

Different rules exist to map binary → DNA, which increases flexibility.

---

### 🔹 Dynamic DNA Rule Switching

Instead of using a fixed rule, the system:

* **Dynamically changes DNA encoding rules**
* Makes encryption more secure
* Prevents pattern detection

---

### 🔹 Entropy

Entropy measures randomness in data:

* High entropy → more secure
* Used to decide **when to switch DNA rules**

---

### 🔹 Chaotic Map

Chaotic systems generate highly unpredictable sequences:

* Used to create **random keys**
* Ensures strong encryption

---

## 🔍 Methodology (Step-by-Step)

1. Input image is selected
2. Secret data is converted into binary
3. Binary data is encoded into DNA sequences
4. Entropy of data is calculated
5. Chaotic map generates random values
6. Based on entropy + chaos, DNA rules are dynamically switched
7. Encoded data is embedded into the image (steganography)
8. Final output → **Secure stego image**
